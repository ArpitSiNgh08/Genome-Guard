1. **For Explorer screenshots:** Visit the URLs above and take full-page screenshots
2. **For wallet:** Connect Petra wallet in the app and screenshot the connection UI
3. **For web app:** Navigate to each page and capture the interface
4. **For CLI:** Run commands and screenshot terminal output

## Image Requirements

- Resolution: Minimum 1920x1080
- Clear and readable text
- Show complete UI/information
